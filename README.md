# Análise Exploratória de Dados de Vendas

## Descrição

Este projeto apresenta uma análise exploratória de um conjunto de dados de vendas utilizando Python, Pandas e Matplotlib.
O objetivo é explorar os dados, identificar padrões, gerar estatísticas descritivas e responder a perguntas de negócio por meio de visualizações e análises.

## Objetivos

- Calcular o faturamento total.
- Calcular o lucro total.
- Identificar a categoria com maior volume de vendas.
- Identificar o produto com maior faturamento.
- Analisar o desempenho das vendas por estado.
- Identificar o cliente que realizou a maior compra.
- Verificar a forma de pagamento mais utilizada.
- Identificar possíveis outliers na coluna de vendas.

## Ferramentas utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

## Estrutura do projeto

- Carregamento e inspeção dos dados.
- Tratamento dos tipos de dados.
- Estatísticas descritivas.
- Análise exploratória.
- Criação de gráficos.
- Identificação de outliers.
- Conclusões sobre os resultados obtidos.
